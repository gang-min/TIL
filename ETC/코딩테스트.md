# 코딩 테스트 유형 및 플랫폼

## `왜 코딩 테스트를 보는가?`

- 문제 해결 능력 + 구현 능력 -> SW 기본 역량(정량적 평가)

## `대표 문제 유형 3가지`

![코딩테스트](/image/대표유형.png)

- 구현

  : 코딩을 통해 문제 요구 사항을 해결하며 답을 도출하는 유형

- 완전 탐색(DFS/BFS)

  : 컴퓨터의 빠른 계산 능력을 기반으로 해서 가능한 경우의 수를 모두 찾아서 최적의 해 또는 올바른 해를 도출하는 유형

- 탐욕 알고리즘

  : 당장 최적인 답을 선택하면서 그 과정을 지속적으로 반복하면서 답을 도출하는 유형
